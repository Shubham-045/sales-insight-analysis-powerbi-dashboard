# Sales Insights – Power BI Dashboard

## 📌 Project Overview
Sales Insights is a data analytics project designed to evaluate sales performance, customer behavior, and revenue trends.  
The project leverages a relational sales database as the data source and presents insights through an interactive **Power BI dashboard**.

The objective is to convert raw sales data into meaningful insights that support data-driven business decisions.

## 🛠 Tools & Technologies Used
- **Power BI Desktop** – Data visualization and dashboard creation  
- **SQL Server / MySQL** – Data storage and querying  
- **DAX (Data Analysis Expressions)** – Calculations and measures  
- **GitHub** – Version control and project sharing  

---

## 📂 Data Source
The dashboard is built on a structured sales dataset consisting of the following key tables:

### Main Tables:
- Customers  
- Products  
- Transactions  
- Markets  
- Date  

The dataset includes details such as:
- Sales amount  
- Order information  
- Product categories  
- Customer data  
- Regional markets  

Data was imported into Power BI and modeled using relationships to enable effective analysis.

---

## 📊 Dashboard Features

The dashboard provides the following insights:

### Key Metrics:
- Total Revenue  
- Total Sales Quantity  
- Revenue by Market  
- Top Customers  
- Top Products  
- Sales Trends Over Time  

### Visualizations:
- Line charts for revenue trends  
- Bar charts for market performance  
- KPI cards for summary metrics  
- Interactive filters and slicers  

---

## 🚀 Key Insights

- Identified top-performing markets contributing the highest revenue  
- Highlighted high-value customers driving significant sales  
- Analyzed regions with declining and growing sales trends  
- Observed product-level demand patterns  
- Tracked monthly and yearly revenue variations  

These insights help businesses make informed decisions and optimize sales strategies.
